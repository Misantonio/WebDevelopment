SVG Drawing

To draw shapes with SVG, you first need to create an SVG element tag with two attributes:width and height.

<svg width=1000 height=1000></svg>

Circle

To create a circle, add a <circle> tag: 

    <svg width=2000 height=2000>
        <circle cx=80 cy=80 r=50 fill=#00FF00 style="stroke:blue;stroke-width:3"/>
    </svg>

Result

Rectangle

<rect> defines a rectangle: 

    <svg width=2000 height=2000>
        <rect width=300 height=100 x=20 y=20 fill=green style="stroke:#FF0000; stroke-width:3"/>
    </svg>

Result
